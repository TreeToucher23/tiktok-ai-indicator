# TikTok "AI" Trading Indicator — Explained

I kept seeing people selling "AI indicators" on TikTok.

I got access to one and checked the source code.

It is NOT AI.

It’s just:
- EMA crossovers
- RSI filters
- VWAP / volume filters

Nothing predictive — just standard lagging indicators with good marketing.

I cleaned up the code and made this open source so people can:
- understand what they’re buying
- stop getting misled
- improve it themselves

## Notes
- Works best in trending markets
- Will fail in sideways / chop (like most EMA systems)
